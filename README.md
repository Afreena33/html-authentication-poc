# HTML Authentication System POC

This project is a simple authentication system built using HTML and styled using Bootstrap 5 and custom CSS.

## Pages Included
- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features
- Responsive design using Bootstrap
- Clean UI with card layout
- Styled forms and buttons
- Navigation using anchor tags
- Custom CSS styling (styles.css)

## Technologies Used
- HTML
- Bootstrap 5
- CSS
